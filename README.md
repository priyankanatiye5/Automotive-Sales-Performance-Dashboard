# Automotive-Sales-Performance-Dashboard
1) Short Description
Automotive Sales Performance Dashboard in Power BI to analyze business performance across multiple dimensions using structured transactional sales data. The objective of the project is to provide a clear, interactive view of sales trends, pricing patterns, product mix contribution, and regional performance using time intelligence calculations.

2)Tech Stack
•  Power BI Desktop 
•  Power Query – Data transformation and cleaning layer for reshaping and preparing the data.
•  DAX (Data Analysis Expressions) – Used for calculated measures, dynamic visuals, and conditional logic.
•  Data Modeling – Relationships established among cars sales table and date table to enable cross-filtering and aggregation.

3)Data Source
Video uploaded by Data Tutorial 
Dataset containing 24,000+ transactional records. The dataset captured detailed information for each car sold, including customer demographics, pricing details, vehicle specifications, dealer information, and order dates.The data included attributes such as customer name, gender, salary, car model, company name, body type, engine type, sales price, order date, dealer name, dealer region, and contact details. This enabled comprehensive analysis across customer behavior, product segmentation, pricing trends, and regional sales performance.
